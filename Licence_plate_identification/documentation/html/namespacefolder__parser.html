License plate identification
This Python script leverages Optical Character Recognition (OCR) to extract text from images containing car license plates. The extracted text is then used to determine the nationality of the car based on the license plate information. The results are saved in a CSV (Comma Separated Values) file for further analysis or integration with other systems.

analyze_folder ( folder_path) 
Analyze the contents of a folder and extract filenames of images.

This function scans the specified folder for image files with common extensions (.jpg, .jpeg, .png, .bmp).
It then returns a list of filenames of images found in the folder.

:param folder_path: Path to the folder containing images.
:type folder_path: str
:return: A list of filenames of images found in the folder.
:rtype: list[str]

:raises OSError: If the specified folder does not exist or cannot be accessed.
